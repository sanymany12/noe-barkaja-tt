# Nem funkcionális követelmények

#### Termék követelmények

A program alacsony erőforrásigényű, nem jelent jelentős terhelést egy mai processzor és videókártya számára. Ideális hardver specifikáció: Intel core i5 generációs processzor vagy jobb, NVIDIA GTX 960 vagy jobb video kártya, 8GB RAM. Hálózati kapcsolatot nem igényel.

#### Megbízhatóság

Megfelelő bemenetek használata mellett nem fordulhat elő hibajelenség, amennyiben azonban a bemenet nem megfelelő, a felhasználó hibaüzenetet kap.

#### Biztonság

A program személyes adatokat nem gyűjt, így nem releváns.

#### Hordozhatóság

Nem igényel külön telepítést, azonnal használható, a legtöbb személyi számítógépen használható.

#### Felhasználhatóság

Felhasználóbarát felhasználói felület, a súgó a játék bármely pontján elérhető, így ha a játékosnak kérdése akad, mindig hozzáfér.

#### Környezeti igények

Nincsenek, önálló program.

#### Működési igények

A játékos számos órán át játszhatja a játékot, futási ideje pár perctől akár órákig is könnyedén terjedhet.

#### Fejlesztési követelmények

* Programnyelv: Java
* Fejlesztőszoftver: IntelliJ IDEA
* Verziókezelő rendszer: Git
* Elsődleges programozási paradigma: objektumorientált programozás
